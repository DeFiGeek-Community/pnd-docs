# Pool Overview

### Base Assets and Collateral Assets

Each pool has one base asset paired with multiple collateral assets, which can be confirmed on the pool list page and pool detail page.

### Example: CJPY

For example, the CJPY pool has CJPY as the base asset, and TXJP, wstETH, USDC, crvUSD, UNI, CRV, PND, and scrvUSD as collateral assets.

### Collateral Asset Tooltip

When you hover over the collateral asset icons in the pool list, all collateral asset names are displayed in a tooltip.
