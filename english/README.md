# punodwoɔ

### What is [punodwoɔ](https://pnd.defigeek.xyz/)

punodwoɔ is a protocol that enables peer-to-peer lending without intermediaries (third parties). Users can borrow base assets by depositing collateral assets into pools, and earn interest by depositing base assets. Anyone can use it, so please review the [protocol details](specification/) and [usage guide (User Guide)](user-guide/) before using.

### Current Base Assets

CJPY, USDC, crvUSD, WETH

### Current Collateral Assets

TXJP, wstETH, USDC, crvUSD, CJPY, UNI, CRV, PND, scrvUSD
